# Polaprint

A native iOS and macOS app for printing photos to Fujifilm Instax printers over WiFi.

## Features

- Direct WiFi printing to Instax SP-1, SP-2, and SP-3 printers
- Live image editor with pan, zoom, and rotation
- Real-time printer status (battery, prints remaining)
- Camera capture and photo library support
- Native SwiftUI interface for iOS and macOS

## Requirements

- iOS 16+ / macOS 13+
- Xcode 15+
- [InstaxKit](../InstaxKit) (included as local package)

## Building

Open `Polaprint.xcodeproj` in Xcode and build for your target device.

## Usage

1. Connect your iPhone/Mac to the Instax printer's WiFi network
2. Launch Polaprint
3. Select or capture a photo
4. Adjust framing using pinch and drag gestures
5. Tap the print button
